## 1. 프로젝트 개요

### 1.1 서비스 소개

**커밍스(Comings)**는 카페, 소규모 상점을 위한 선결제(예치금) 관리 플랫폼이다. 기존 수기 장부나 스프레드시트로 관리하던 선불 충전 고객의 잔액을 디지털화하여, 실시간 잔액 조회, 간편 충전/차감, 거래 내역 관리, 매출 분석까지 하나의 시스템에서 해결한다.

### 1.2 브로셔 목적

- 카페 및 소규모 상점 사장님 대상 오프라인 홍보물
- 수기 장부 관리의 불편함(페인 포인트)을 시각적으로 부각하고, 커밍스 도입 시 편의성을 대비하여 보여줌
- 무료 가입 유도 및 서비스 인지도 확보

### 1.3 타겟 독자

- **1차 타겟:** 카페 사장님 (개인 카페, 동네 카페)
- **2차 타겟:** 소규모 상점 운영자 (네일샵, 필라테스, 세탁소 등 선불 충전 방식을 쓰는 업종)
- **연령대:** 30~50대
- **특성:** 디지털 친숙도 중간, 직관적이고 쉬운 설명 선호

### 1.4 톤 앤 매너

| 항목 | 지침 |
|------|------|
| **화법** | 존댓말, 친근하되 신뢰감 있는 톤 |
| **어휘** | 기술 용어 배제, 일상 용어 사용 |
| **감성** | 따뜻하고 아날로그 감성 (카페 분위기) |
| **핵심 메시지** | "수기 장부의 불편함 → 커밍스로 간편하게" |

## 2. 브로셔 사양 (물리적 스펙)

### 2.1 접지 형태

**트라이폴드(Tri-fold)** - 3단 접지 브로셔

```
[펼친 상태 - 앞면 (Outside)]

  ┌──────────────┬──────────────┬──────────────┐
  │              │              │              │
  │   패널 A     │   패널 B     │   패널 C     │
  │  (뒷면)      │  (표지 뒷면)  │  (표지/앞면)  │
  │              │              │              │
  │  Page 3      │  (접히는 면)  │  Page 1      │
  │              │              │              │
  └──────────────┴──────────────┴──────────────┘

[펼친 상태 - 뒷면 (Inside)]

  ┌──────────────┬──────────────┬──────────────┐
  │              │              │              │
  │   패널 D     │   패널 E     │   패널 F     │
  │  (안쪽 왼)   │  (안쪽 중앙)  │  (안쪽 오른)  │
  │              │              │              │
  │  Page 2-L    │  Page 2-C    │  Page 2-R    │
  │              │              │              │
  └──────────────┴──────────────┴──────────────┘
```

### 2.2 용지 규격

| 항목 | 사양 |
|------|------|
| **펼친 크기** | 297mm (가로) x 210mm (세로) - A4 가로 방향 |
| **접힌 크기** | 99mm (가로) x 210mm (세로) |
| **패널 너비** | 좌측 패널(A/D): 98mm, 중앙 패널(B/E): 100mm, 우측 패널(C/F): 99mm |
| **패널 높이** | 210mm |
| **재단 여백(Bleed)** | 사방 3mm |
| **실제 작업 사이즈** | 303mm x 216mm (재단선 포함) |

> **참고:** 안쪽으로 접히는 패널(패널 B)은 접힘 간섭 방지를 위해 1~2mm 좁게 설정할 수 있다 (98mm).

### 2.3 접지 방식 상세

```
접는 순서:
1. 패널 B(표지 뒷면)를 안쪽으로 접는다
2. 패널 C(표지)가 가장 바깥에 위치한다
3. 펼치면 패널 D-E-F(안쪽 3면)가 하나의 큰 화면으로 보인다
```

### 2.4 안전 영역 (Safe Zone)

| 영역 | 규격 |
|------|------|
| **외곽 안전 마진** | 재단선에서 안쪽으로 5mm |
| **접지선 안전 마진** | 접지선 좌우 각 5mm (총 10mm) |
| **텍스트 영역** | 안전 마진 안쪽 |
| **이미지/배경** | 재단선(Bleed) 바깥까지 확장 가능 |
