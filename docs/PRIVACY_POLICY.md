## 1. 개인정보의 처리 목적

회사는 "커밍스(Comings)" 서비스(https://www.comings.co.kr, 이하 "서비스")의 제공을 위하여 다음의 목적으로 개인정보를 처리합니다. 처리하고 있는 개인정보는 다음의 목적 이외의 용도로는 이용되지 않으며, 이용 목적이 변경되는 경우에는 「개인정보 보호법」 제18조에 따라 별도의 동의를 받는 등 필요한 조치를 이행할 예정입니다.

### 1.1 회원 가입 및 관리

회원 가입 의사 확인, 회원제 서비스 제공에 따른 본인 식별·인증, 회원자격 유지·관리, 서비스 부정이용 방지, 중복 가입 방지, 각종 고지·통지 목적으로 개인정보를 처리합니다.

### 1.2 서비스 제공

선결제(예치금) 고객 관리, 충전·차감 거래 기록, 잔액 조회, 거래 내역 관리, 매출 통계 분석, 데이터 내보내기(CSV) 등 서비스 제공 목적으로 개인정보를 처리합니다.

### 1.3 사업자 인증

사업자등록번호를 통한 사업자 확인 및 중복 가입 방지 목적으로 개인정보를 처리합니다.

### 1.4 보안 및 서비스 안정성 확보

비정상적인 접근 탐지, 로그인 시도 제한, 부정 이용 방지, 서비스 오류 모니터링 및 개선 목적으로 개인정보를 처리합니다.

### 1.5 비밀번호 재설정

회원의 비밀번호 분실 시 본인 확인 및 비밀번호 재설정 안내 이메일 발송 목적으로 개인정보를 처리합니다.

## 2. 수집하는 개인정보의 항목 및 수집 방법

### 2.1 수집하는 개인정보 항목

#### 가. 회원 가입 시 (필수)

| 항목 | 설명 |
|------|------|
| **아이디(사용자명)** | 로그인 식별자 (4~20자, 영문 소문자 및 숫자) |
| **비밀번호** | 서비스 로그인용 (8자 이상, 영문+숫자 조합, 암호화 저장) |
| **상호명** | 매장/사업체 이름 |
| **사업자등록번호** | 사업자 확인 및 중복 가입 방지 용도 (10자리, 국세청 진위 확인) |
| **보안 PIN** | 4자리 숫자, 민감 작업(고객 수정·삭제, 거래 취소 등) 보호용 (암호화 저장) |

#### 나. 회원 가입 시 (선택)

| 항목 | 설명 |
|------|------|
| **이메일 주소** | 비밀번호 재설정 안내 수신용 |

#### 다. 서비스 이용 과정에서 회원이 입력하는 고객 정보

회원(매장 운영자)이 자신의 선결제 고객을 관리하기 위해 직접 입력하는 정보로, 회원이 해당 정보의 수집·관리에 대한 책임을 집니다.

| 항목 | 설명 |
|------|------|
| **고객 이름** | 선결제 이용 고객의 이름 (1~50자) |
| **고객 전화번호** | 전체 전화번호 및 뒷자리 4자리 (고객 식별·검색 용도) |
| **잔액 정보** | 선결제 충전 잔액 |
| **거래 내역** | 충전·차감·취소 기록 (금액, 결제수단, 메모, 일시) |

#### 라. 서비스 이용 과정에서 자동으로 수집되는 정보

| 항목 | 설명 |
|------|------|
| **IP 주소** | 로그인 시도 제한 및 부정 이용 방지 목적 (메모리 임시 저장, 로그에는 일부만 기록) |
| **접속 토큰(JWT)** | 로그인 세션 유지용 (브라우저 저장소에 저장) |
| **서비스 오류 정보** | 오류 발생 시 오류 메시지, 스택 트레이스, 매장 식별자(개인정보 미포함) |

### 2.2 개인정보 수집 방법

| 수집 방법 | 해당 항목 |
|-----------|----------|
| **회원 가입 화면** | 아이디, 비밀번호, 상호명, 사업자등록번호, 보안 PIN, 이메일(선택) |
| **Cloudflare Turnstile** | 봇 방지 토큰 (회원 가입 시 자동) |
| **서비스 이용 중 회원 입력** | 고객 이름, 고객 전화번호, 거래 내역 |
| **자동 수집** | IP 주소, 접속 토큰, 서비스 오류 정보 |

### 3.1 회원 정보

## 4. 개인정보의 제3자 제공

회사는 정보주체의 개인정보를 제1조에서 명시한 범위 내에서만 처리하며, 정보주체의 동의, 법률의 특별한 규정 등 「개인정보 보호법」 제17조 및 제18조에 해당하는 경우에만 개인정보를 제3자에게 제공합니다.

**현재 개인정보를 제3자에게 제공하고 있지 않습니다.**

회사는 원활한 서비스 제공을 위해 다음과 같이 개인정보 처리업무를 위탁하고 있습니다.

| 수탁업체 | 위탁 업무 내용 | 전달되는 정보 | 보유 기간 |
|---------|--------------|-------------|----------|
| **Supabase Inc.** | 데이터베이스 호스팅 및 관리 | 회원 정보, 고객 정보, 거래 내역 전체 | 위탁 계약 종료 시까지 |

| **Resend Inc.** | 이메일 발송 (비밀번호 재설정) | 이메일 주소, 재설정 코드 | 발송 완료 후 Resend 정책에 따름 |

## 6. 개인정보의 파기절차 및 방법

### 6.1 파기절차

회사는 개인정보 보유 기간의 경과, 처리 목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체 없이 해당 개인정보를 파기합니다.

정보주체로부터 동의받은 개인정보 보유 기간이 경과하거나 처리 목적이 달성되었음에도 불구하고 다른 법령에 따라 개인정보를 계속 보존하여야 하는 경우에는, 해당 개인정보를 별도의 데이터베이스(DB)로 옮기거나 보관장소를 달리하여 보존합니다.

### 6.2 파기방법

| 정보 유형 | 파기 방법 |
|----------|----------|
| **전자적 파일 형태** | 기록을 재생할 수 없는 기술적 방법을 사용하여 삭제 |
| **종이에 기록된 개인정보** | 해당 사항 없음 (전자적 처리만 수행) |
| **데이터베이스 기록** | 해당 레코드를 완전 삭제 (DELETE) |
| **암호화된 정보 (비밀번호, PIN)** | 해시값 삭제 (원본 복원 불가) |
| **접속 토큰** | 토큰 블랙리스트 등재 후 만료 시 자동 삭제 |
| **메모리 임시 정보 (IP 주소 등)** | 서버 재시작 시 자동 삭제 또는 설정 시간 경과 후 삭제 |

## 7. 정보주체의 권리·의무 및 행사 방법

### 7.1 정보주체의 권리

정보주체는 회사에 대해 언제든지 다음 각 호의 개인정보 보호 관련 권리를 행사할 수 있습니다.

1. **개인정보 열람 요구**
2. **오류 등이 있을 경우 정정 요구**
3. **삭제 요구**
4. **처리 정지 요구**

### 7.2 권리 행사 방법

| 방법 | 상세 |
|------|------|
| **서비스 내 직접 처리** | 상호명 변경, 비밀번호 변경, PIN 변경, 이메일 변경은 서비스 설정 화면에서 직접 가능 |
| **고객 데이터 관리** | 회원이 입력한 고객 정보(이름, 전화번호)는 서비스 내에서 직접 수정·삭제 가능 |
| **데이터 내보내기** | 고객 목록을 CSV 파일로 내보내기 가능 (서비스 내 기능) |
| **이메일 요청** | 계정 삭제, 개인정보 열람·정정·삭제·처리 정지 등은 [이메일 주소]로 요청 |

### 7.3 유의사항

- 권리 행사는 서면, 전자우편 등을 통하여 하실 수 있으며, 회사는 이에 대해 지체 없이 조치하겠습니다.
- 정보주체가 개인정보의 오류 등에 대한 정정 또는 삭제를 요구한 경우에는 회사는 정정 또는 삭제를 완료할 때까지 당해 개인정보를 이용하거나 제공하지 않습니다.
- 권리 행사는 정보주체의 법정대리인이나 위임을 받은 자 등 대리인을 통하여 하실 수도 있습니다. 이 경우 「개인정보 보호법」 시행규칙 별지 제11호 서식에 따른 위임장을 제출하셔야 합니다.
- 개인정보 열람 및 처리 정지 요구는 「개인정보 보호법」 제35조 제4항, 제37조 제2항에 의하여 정보주체의 권리가 제한될 수 있습니다.

## 9. 개인정보 자동 수집 장치의 설치·운영 및 거부

### 9.1 쿠키(Cookie)

회사는 인증 목적의 쿠키를 사용하지 않습니다. 로그인 세션 유지를 위해 **브라우저 로컬 스토리지(localStorage)**에 인증 토큰을 저장합니다.

### 9.2 로컬 스토리지(localStorage)

서비스 이용을 위해 브라우저 로컬 스토리지에 다음 정보가 저장됩니다.

| 저장 항목 | 내용 | 목적 |
|----------|------|------|
| `access_token` | JWT 접속 토큰 | 로그인 세션 유지 |
| `auth-storage` | 인증 상태, 매장 ID, 매장명, 다크모드 설정, 마지막 활동 시간 | 서비스 이용 편의 |

### 9.3 거부 방법

- 브라우저 설정에서 로컬 스토리지를 삭제하거나 사이트 데이터를 삭제할 수 있습니다.
- 로컬 스토리지 데이터를 삭제하면 로그인 상태가 해제되며, 다시 로그인이 필요합니다.

### 9.4 Cloudflare Turnstile

회원 가입 시 봇 방지를 위해 Cloudflare Turnstile 서비스가 작동합니다. 이 과정에서 Cloudflare의 자체 쿠키 및 스크립트가 실행될 수 있으며, 이는 Cloudflare의 개인정보처리방침에 따릅니다.

### 9.5 Sentry (오류 모니터링)

서비스 안정성 개선을 위해 Sentry 오류 모니터링 서비스를 사용합니다. 브라우저에서 오류 발생 시 기술적 정보(오류 메시지, 브라우저 정보)가 수집될 수 있으나, 개인정보 전송은 비활성화되어 있습니다.
