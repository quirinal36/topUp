## 스토리보드

### SCENE 0 — 인트로 (0:00 ~ 0:05) [5초]

```
┌─────────────────────────────────────────────────┐
│                                                 │
│              ☕ 커밍스                            │
│                                                 │
│         선결제 관리, 쉽고 빠르게                    │
│                                                 │
│      ─── 회원가입 가이드 ───                      │
│                                                 │
│                                                 │
└─────────────────────────────────────────────────┘
```

**화면 구성:**
- 배경: 따뜻한 브라운 → 크림 그라데이션 (앱 다크모드 배경색 활용)
- 중앙에 커밍스 로고(☕) 페이드인
- "커밍스" 텍스트 타이핑 효과로 등장
- 하단에 "회원가입 가이드" 슬라이드업

**나레이션 (자막):**
> 커밍스로 선결제 관리를 시작해보세요.
> 회원가입은 간단한 4단계로 완료됩니다.

**모션:**
1. (0.0s) 배경 그라데이션 서서히 밝아짐
2. (0.5s) ☕ 로고 스케일업 + 페이드인
3. (1.5s) "커밍스" 텍스트 등장
4. (2.5s) "회원가입 가이드" 슬라이드업
5. (4.5s) 전체 페이드아웃 → Scene 1로 전환

**배경음악:** 여기서 시작 (부드러운 도입부)

**디자인 포인트:**
- 타이핑 중인 필드에 하이라이트 효과 (주목 유도)

- "다음" 버튼 클릭 시 부드러운 프레스 애니메이션
